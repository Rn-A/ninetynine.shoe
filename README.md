# Ninetynine Shoe - Premium Shoe Care

Website resmi Ninetynine Shoe untuk layanan cuci sepatu premium di wilayah Cilacap dan Purwokerto.

## Fitur Utama

- **Landing Page Premium**: Informasi layanan, harga, dan testimoni pelanggan.
- **Sistem Pemesanan**: Pelanggan bisa melakukan booking layanan secara online.
- **Pelacakan Pesanan**: Cek status cuci sepatu menggunakan nomor resi.
- **Admin Dashboard**: Kelola layanan, pesanan, testimoni, dan galeri secara dinamis.
- **Pick Up & Delivery**: Integrasi informasi layanan antar jemput.

## Cara Menjalankan Project

### Prerequisites
- Node.js installed

### 1. Backend Setup
1. Masuk ke folder `backend`: `cd backend`
2. Install dependencies: `npm install`
3. Jalankan server: `npm run dev` (Server berjalan di http://localhost:5001)

### 2. Frontend Setup
1. Masuk ke folder `frontend`: `cd frontend`
2. Install dependencies: `npm install`
3. Jalankan aplikasi: `npm run dev` (Aplikasi berjalan di http://localhost:3000)

## Tech Stack
- **Frontend**: Next.js 15, Tailwind CSS, Lucide Icons, Font Awesome.
- **Backend**: Express.js, Sequelize (SQLite/MySQL).
- **Database**: SQLite (default untuk development).
