# Ninetynine Shoe Frontend

Bagian frontend dari aplikasi Ninetynine Shoe yang dibangun menggunakan Next.js.

## Struktur Project

- `src/app`: Menggunakan App Router Next.js untuk halaman utama dan dashboard admin.
- `src/context`: State management menggunakan React Context.
- `src/old`: Berisi file-file template awal yang digunakan untuk desain premium.
- `public/uploads`: Lokasi penyimpanan file gambar dan video dari CMS.

## Development

Jalankan perintah berikut untuk memulai mode development:

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser untuk melihat hasilnya.

## Build for Production

```bash
npm run build
npm start
```
